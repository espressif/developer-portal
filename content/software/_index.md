## Zephyr

Zephyr is an open-source real-time operating system that supports Espressif devices and is well suited for scalable embedded projects that benefit from a modular RTOS architecture.

### Support Status for Zephyr

Check the current Zephyr support status for Espressif chips, including platform availability and version-specific updates.

## Rust

Rust brings memory safety and modern language features to embedded development on Espressif platforms, making it a strong choice for reliable and maintainable applications.

For more details, read the official [Rust on ESP book](https://esp-rs.github.io/book/) and the [Awesome ESP Rust](https://github.com/esp-rs/awesome-esp-rust).
